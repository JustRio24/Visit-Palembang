## Visit Palembang

A modern, responsive web application to promote the beauty, culture, and tourist attractions of **Palembang**, Indonesia.  
Created as part of the **Web Design Competition 2025**.

## Purpose

This website aims to showcase the city of Palembang as a top-tier travel destination to a global audience.  
All design, content, and code were developed **manually** without the use of generative AI tools.

## Tech Stack

- **Framework**: React 19
- **Styling**: Tailwind CSS v4, Material Tailwind
- **Routing**: React Router v7
- **Animation**: Framer Motion
- **Carousel**: Swiper.js
- **Gallery Layout**: React Masonry
- **Image Viewer**: Yet Another React Lightbox

- Masonry gallery with lightbox preview
- Interactive carousel for featured destinations
- Parallax scrolling sections
- Newsletter form with validation

- Google Fonts integration (`Dancing Script`, `Inter`)

## Setup & Run Locally

1. Clone this repository or unzip the project
2. Install the dependencies:

   npm install

3. Run the app in development mode:

   npm run dev

4. Open http://localhost:5173/visit-palembang/#/ in your browser

- Deployment
  This project can be deployed to GitHub Pages using:

npm run deploy
(Requires configuration with gh-pages and correct homepage in package.json)
